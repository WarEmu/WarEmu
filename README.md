  ## Contribution Guidelines

We appreciate anyone who would like to contribute to the program. Please take a moment to read the following guidelines before contributing:
### List of Contributions
- Bug Fixes
- Feature Development
- Enhancements
- Documentation
- Testing
### How to Contribute:
1. Fork the Repository
  - Fork the repository of WarEmu to your GitHub account
2. Clone the Repository
  - Go into any command line and use the following code
````
  git clone https://github.com/WarEmu/WarEmu.git
````
3. Create a Branch
   - Create a new branch for all of your future contributions
````
  git checkout -b <your-branch-name>
````
4. Make Changes
5. Stage Changes
   - Add all of the changes you want to submit
````
  git add <changed-file-name>
````
6. Commit your changes
````
  git commit -m "Add commit message here"
````
7. Push Changes
````
  git push origin <your-branch name>
````
8. Open Pull Request
   - Submit a pull request from your branch to the update branch. Our team will review the contribution, provide you with feedback if necessary, and merge if it meets our standards.
I wish you all good luck and thank you for trying to make WarEmu a better program!
